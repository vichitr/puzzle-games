# @puzzle-games/types

Framework-agnostic TypeScript interfaces for puzzle JSON used by [Puzzle Games](https://puzzlegames.fun).

These types mirror the on-disk format documented in [`docs/puzzle-data-format.md`](../../docs/puzzle-data-format.md). They have no runtime dependencies and can be copied or imported into any TypeScript project.

```ts
import type { QueensPuzzle, PuzzleType } from "./index";
```
